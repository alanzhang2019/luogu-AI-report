## 1. 核心机制：GESP 4 条规则

### 1.1 规则总览

| # | 规则 | 触发条件 | 产品机制 |
|---|---|---|---|
| 1 | **必须从 1 级开始** | 新学员首次考试 | `gesp_exams.registered_level >= 1`，首考 = 1 |
| 2 | **90+ 可跳一级** | 分数 ≥ 90 | 下次可报 `level + 2`（封顶 8） |
| 3 | **跳级失败重考原级** | 分数 < 60 | 下次仍报 `last.registered_level` |
| 4a | **7 级 80+ 免 CSP-J 初赛** | GESP 7 级 ≥ 80 | `exempts_csp_j = true` |
| 4b | **8 级 60+ 免 CSP-J 初赛** | GESP 8 级 ≥ 60 | `exempts_csp_j = true` |
| 4c | **8 级 80+ 免 CSP-S 初赛** | GESP 8 级 ≥ 80 | `exempts_csp_s = true` |

### 1.2 "下次可报"算法

```python
GESP_MAX_LEVEL = 8
GESP_PASS_SCORE = 60
GESP_SKIP_SCORE = 90

def next_eligible_gesp_level(last_exam):
    """
    根据最近一次 GESP 考试结果，计算下次可报等级。
    last_exam: gesp_exams row or None
    """
    if last_exam is None:
        return 1
    if last_exam.actual_score < GESP_PASS_SCORE:
        return last_exam.registered_level     # 重考原级
    if last_exam.actual_score >= GESP_SKIP_SCORE:
        return min(last_exam.registered_level + 2, GESP_MAX_LEVEL)  # 跳 1 级
    return last_exam.registered_level + 1     # 正常升 1 级
```

### 1.3 "免初赛"算法

```python
GESP_EXEMPTION_MATRIX = {
    # GESP 等级 → {分数阈值: [可免的初赛]}
    7: {
        80: ["csp_j"],       # 7 级 80+ → 免 J
    },
    8: {
        60: ["csp_j"],       # 8 级 60+ → 免 J
        80: ["csp_j", "csp_s"],  # 8 级 80+ → 免 J + S
    },
}

def compute_exemptions(gesp_level, score):
    """根据 GESP 等级和分数，返回可免的初赛列表。"""
    if gesp_level not in GESP_EXEMPTION_MATRIX:
        return []
    exemptions = []
    for threshold in sorted(GESP_EXEMPTION_MATRIX[gesp_level].keys(), reverse=True):
        if score >= threshold:
            exemptions.extend(GESP_EXEMPTION_MATRIX[gesp_level][threshold])
    return sorted(set(exemptions))
```

### 1.4 段位图规则

```
[1★]──[2★]──[3★]──[4★]──[5★]──[6✦]──[7 ]──[8 ]
  ✓    ✓    ✓    ✓    ✓    ✓   92分 
                              │
                       触发跳级 + 免 CSP-J
```

- ` `（空格）= 未通过
- `★` = 已通过（60-89）
- `✦` = 优秀通过（80+）→ 解锁对应免初赛
- 已通过 7 级 / 8 级的位置额外标注 `J 免` / `S 免`

### 1.5 跳级决策树（家长辅助，AI 推荐）

```
              学员 GESP N 级 K 分
                       │
         ┌─────────────┼─────────────┐
         ▼             ▼             ▼
     A. 报 N+1     B. 报 N+2     C. 报 N+3
      (稳)         (跳 1 级)     (跳 2 级)
         │             │             │
   通过率 ~95%    通过率 ~70%   通过率 ~25%
   时间表正常     节省 1 次考试 节省 2 次考试
                               但失败重考 N+2
         │             │             │
         ▼             ▼             ▼
       ✅ 适合       ⚠️ 中等       ❌ 不建议
       "基础不       "DP 入门      "跨度
        错"          已会可冲"      太大"
```

> AI 推荐：B（跳 1 级）为默认；A 用于"近期状态差"；C 用于"实力远超当前级别"。

```sql
-- 1. 赛事主表（年度数据，CCF 12 月公布下一年）
CREATE TABLE competitions (
  id              INTEGER PRIMARY KEY AUTOINCREMENT,
  code            TEXT UNIQUE NOT NULL,    -- 'GESP-2026-03-L5'
  name            TEXT NOT NULL,
  type            TEXT NOT NULL,           -- 'gesp' / 'csp_j1' / 'csp_s1' / 'csp_j2' / 'csp_s2' / 'noip' / 'noi' / 'wc' / 'apio' / 'provincial' / 'qiangji' / 'zk_zhongkao' / 'gaokao'
  level           INTEGER,                 -- GESP 1-8 / 0=无级别
  exam_date       DATE NOT NULL,
  registration_deadline DATE,
  location        TEXT,
  target_audience TEXT,                   -- '小学 5-6 年级' / '初中' / '高中' / '强基' / '中考'
  fee_cny         INTEGER DEFAULT 0,
  source_url      TEXT,
  data_year       INTEGER NOT NULL,
  notes           TEXT,
  updated_at      DATETIME DEFAULT CURRENT_TIMESTAMP
);
CREATE INDEX idx_comp_date ON competitions(exam_date);
CREATE INDEX idx_comp_type_year ON competitions(type, data_year);

-- 2. 学员 vs 赛事 关联
CREATE TABLE student_competitions (
  id              INTEGER PRIMARY KEY AUTOINCREMENT,
  student_id      INTEGER NOT NULL,
  competition_id  INTEGER NOT NULL,
  registered      BOOLEAN DEFAULT 0,
  target_score    INTEGER,
  target_rank     TEXT,
  actual_score    INTEGER,
  actual_rank     TEXT,
  result_level    TEXT,                    -- 'passed' / 'failed' / 'absent' / 'unknown'
  notes           TEXT,
  created_at      DATETIME DEFAULT CURRENT_TIMESTAMP,
  UNIQUE(student_id, competition_id)
);

-- 3. 学员 GESP 考试记录（hard fact）
CREATE TABLE gesp_exams (
  id              INTEGER PRIMARY KEY AUTOINCREMENT,
  student_id      INTEGER NOT NULL,
  exam_id         INTEGER NOT NULL,
  registered_level INTEGER NOT NULL,       -- 报的几级
  actual_score    INTEGER,                 -- 0-100
  passed          BOOLEAN,                 -- >=60
  can_skip_next   BOOLEAN DEFAULT 0,       -- >=90
  exempts_csp_j   BOOLEAN DEFAULT 0,       -- 7级80+ / 8级60+
  exempts_csp_s   BOOLEAN DEFAULT 0,       -- 8级80+
  certificate_no  TEXT,
  notes           TEXT,
  recorded_by     TEXT,                    -- 'admin' / 'self' / 'parent'
  created_at      DATETIME DEFAULT CURRENT_TIMESTAMP,
  UNIQUE(student_id, exam_id)
);
CREATE INDEX idx_gesp_student ON gesp_exams(student_id, registered_level);

-- 4. 政策日历
CREATE TABLE policy_events (
  id              INTEGER PRIMARY KEY AUTOINCREMENT,
  event_code      TEXT UNIQUE,
  name            TEXT NOT NULL,
  category        TEXT,                    -- 'qiangji' / 'zizhao' / 'zongping' / 'baosong' / 'gaokao' / 'zk_zhongkao'
  event_date      DATE,
  target_audience TEXT,
  source_url      TEXT,
  description     TEXT,
  data_year       INTEGER NOT NULL
);
```

## 9. 风险与对冲

| 风险 | 触发 | 对冲 |
|---|---|---|
| **CCF 规则变动** | 免初赛规则改了 | `competitions.json` 标注规则生效日期 + 异常告警 |
| **GESP 跳级失败投诉** | 跳级没过家长投诉 | 决策树必须给 3 选项，跳级不是默认 |
| **冲刺营效果不佳** | 4 周后未达 80 分 | 退 50% + 延长 1 期 |
| **AI 估算 GESP vs 真考偏差** | 估 4 级实际考 2 级 | UI 永远显示"AI 估算"水印 |
| **StudyMate 改架构** | 重构导致跳转失效 | 只调公共 URL + 参数，不绑内部 API |
| **StudyMate 服务挂了** | 部署/网络问题 | 降级 → 跳洛谷题解 |
| **洛谷反爬** | 单 IP 抓取多账号限流 | 错峰 + 多账号池 + `.source_cache` 复用 |
| **PIPL 未成年人合规** | 14 岁以下无授权入库 | `is_minor` 强制 + `guardian_consent_at` |
| **政策日历数据过期** | 强基/中考政策发布后未更新 | UI 显式"最后更新于 YYYY-MM-DD"水印 |
| **同类竞争** | 作业帮/小猿/有道做 OI | 护城河 = OI 数据 + GESP 跳级 + StudyMate AI 讲题独家 |
