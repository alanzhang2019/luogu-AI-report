# AI 作业入口版文案说明

## 文档目的
本文件用于给设计、前端、产品协作使用，定义 `AI错题讲解机` 在 `作业入口版` 场景下的首版页面文案。

当前文案覆盖以下页面：
- 首页
- 上传与识别确认
- 讲解结果页
- 验证页
- 验证结果页
- 我的题
- 给家长看
- 通用状态文案

## 产品定位
- 主入口场景：学生在做当天作业时遇到不会的数学题
- 目标用户：小学数学作业场景，优先 4-6 年级
- 核心表达：不是直接给答案，而是先讲清楚，再用 1-2 道同类题确认是否真的学会

## 文案风格原则
- 句子短，口语化
- 不用技术词，不强调 AI、多智能体、模型
- 不鼓励抄答案
- 先解决当前卡题，再过渡到纠错与验证
- 对孩子友好，对家长可解释

## 页面结构总览
```text
首页
  -> 上传题目页
  -> 识别确认页
  -> 讲解结果页
  -> 验证页
  -> 验证结果页
  -> 我的题
  -> 给家长看
```

## 十、给设计和前端的实现建议
- 结果页请固定为四个卡片区块，不要让后端直接返回任意长段文本。
- 讲解正文建议走结构化字段，前端负责版式排版。
- 首页只保留一个强主入口，避免用户分流。
- 验证页不建议一次超过两题，先降低使用压力。
- 给家长看的页面不要做复杂数据分析，首版只保留 3-4 条结果摘要。
